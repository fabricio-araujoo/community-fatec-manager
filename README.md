## community-fatec-api

### Como executar

```bash
npm install
npm start
```

- Acesse qualquer endpoint do routes com o `http://localhost:8080/` no começo, exemplo: `http://localhost:8080/packageManager/currentUser`

Não se esqueça de enviar os parâmentros certos.
